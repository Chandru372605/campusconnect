# CampusConnect

A modern full-stack community platform for college students.

## Stack
- React, Tailwind CSS (frontend)
- Node.js, Express.js (backend)
- MongoDB Atlas (database)
- Firebase Auth/JWT, Cloudinary, Socket.IO

## Features
- Authentication (including Google OAuth & college email verification)
- Notes sharing
- Lost & Found
- Student Marketplace
- Confessions (anonymous + moderation)
- Placement hub
- Real-time chat & notifications
- Modern UI/UX (mobile-first, dark mode, etc.)
- Admin dashboard

## Setup

1. Clone repo:  
   `git clone https://github.com/Chandru372605/CampusConnect`
2. Install dependencies in `/client` and `/server`
3. Configure `.env` in `/server`
4. Start backend:  
   `cd server && npm run dev`
5. Start frontend:  
   `cd client && npm start`
